# Open Science

This section covers various aspects of open science.

## Output is published with an open license.

**Description**: The research output is published with an open license that allows modification and (commercial) redistribution, like CC-BY.


[Details](https://egonw.github.io/journal-guidances-template/code/JEG000003)

## Metadata about the output is shared as public domain or CCZero

**Description**: Metadata is shared as public domain or CCZero, and includes statements about authorship, copyright ownership, title, publication date, abstract, etc.

Read more in [<a href="#citeref1">1</a><a href="#citeref2">2</a>].

[Details](https://egonw.github.io/journal-guidances-template/code/JEG000004)
